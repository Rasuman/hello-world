<h1>Exponents</h1>
<p>For Example, 2<sup>2</sup> = 4, 3<sup>3</sup> = 27. (2) and (3) are the base and the upper portion of the base are exponent. Consider 2<sup>2</sup>= 2 x 2 = 4 and 3<sup>3</sup> = 3 x 3 x 3 = 27. This is how you solve exponent</p>
<p>_____________________________________________________________________________________________________________________________________________</p>
<h3>Properties of Exponents</h3>
<p>Consider another example:</p>
<p>
(5 x 3)<sup>3</sup> = (5 x 3) x (5 x 3) x (5 x 3) or (3 x 3 x 3) x (5 x 5 x 5) = 5<sup>3</sup> x 3<sup>3</sup></p>
<p><b>Example 1.2</b></p>
<p>Consider 3<sup>3</sup> = 3 x 3 x 3; 3<sup>4</sup> = 3 x 3 x 3 x 3<br>
3<sup>4</sup> x 3<sup>3</sup> = 3 x 3 x 3 x 3 x 3 x 3 x 3 = 3<sup>4+3</sup> = 3<sup>7</sup><br>
Similarity, 4<sup>4</sup> x 4<sup>3</sup> = 4 x 4 x 4 x 4 x 4 x 4 x 4 = 4<sup>4+3</sup> = 4<sup>7</sup>.
</p>
<p><b>Example 1.3</b></p>
<p>2<sup>4</sup> / 2<sup>2</sup> = 4 </p>
<p>Consider this 2<sup>4</sup> / 2<sup>2</sup> = 2<sup>4-2</sup> = 2<sup>2</sup> = 4</p>
