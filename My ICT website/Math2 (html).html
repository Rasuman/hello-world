Definition of Scientific Notation

A number is said to be scientific notation if it is written in the form.

m x 10n

where m is a number greater than or equal to 1 and less than 10, and n is an integer.

The following are examples:

5.3 = 5.3 x 100 (The decimal point in 5.3 must be moved 0 places.)

53 = 5.3 x 101 (The decimal point in 5.3 must be moved 1 places to the right.)

53,000 = 5.3 x 104 (The decimal point in 5.3 must be moved 4 places to the right.)

0.53 = 5.3 x 10-1 (The decimal point in 5.3 must be moved 1 places to the left.)

0.053 = 5.3 x 10-2 (The decimal point in 5.3 must be moved 2 places to the left.)

0.00053 = 5.3 x 10-4 (The decimal point in 5.3 must be moved 4 places to the left.)

Example 1.1

  1. 270,000,000
  2. 0.000 000 000 000 000 000 000 019 9

Solution:

  1. 270,000,000 = 2.7 x 108
  2. 0.000 000 000 000 000 000 000 019 9 = 1.99 x 10-23

Example 1.2

(5 x 104) x (9 x 10-7)

Solution:

(5 x 104) x (9 x 10-7) = (5 x 9) x (104 x 10-7)
= 45 x 104-7
= 45 x 10-3
= (4.5 x 101) x 10-3
= 4.5 x 101-3
= 4.5 x 10-2
